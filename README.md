# Portfel PRO v. 1.0 — Etap 25

Wersja testowa aplikacji Portfel PRO.

## Zmiany w etapie 25

- Dodano zakładkę **Pomoc**.
- Dodano instrukcję obsługi programu.
- Dodano szczegółową instrukcję połączenia z Dropbox.
- Opisano różnice między kopią danych, scalaniem i zastępowaniem danych.
- Opisano działanie tooltipów na komputerze i telefonie.
- Podbito wersję cache do v25.

## Wgrywanie

Wgraj na GitHub Pages całą zawartość paczki ZIP, nie pojedyncze pliki.
Po wgraniu kliknij w programie **Odśwież program** albo otwórz adres z dopiskiem `?v=25`.

## Ważne

Program powinien być uruchamiany przez HTTPS, np. GitHub Pages. Uruchamianie bezpośrednio z pliku `index.html` może ograniczyć PWA, service worker, cache, Dropbox i mikrofon.
